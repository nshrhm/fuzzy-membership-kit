# 診断指標

v0.2.0.dev0 の診断指標は、査読者が実装されたワーピング関数、基底関数、合成メンバーシップ関数を確認しやすくするための数値的チェックです。数学的な導出を置き換えるものではありませんが、設定済みの関数を監査しやすくします。

## 確認する内容

- 範囲: 値が許容誤差内で `[0, 1]` に収まるか。
- 単調性: サンプル値が期待どおり増加または減少するか。
- アンカー誤差: 診断対象の関数に対する目標値からの数値的ずれ。
- 局所識別性: 指定点の近くでの有限差分による傾き。
- 尾部圧縮比: `far` 近傍の局所識別性を `focus` 近傍の局所識別性で割った値。

ワーピング関数 `w` を診断する場合、遠方アンカーの目標はワープ後座標の `q` です。フォーカス-aware メンバーシップ `mu(u)=h(w_theta(u))` を診断する場合、遠方アンカーの目標はメンバーシップ値 `h(q)` です。焦点についても同様に、`w(focus)=0.5` ですが、`mu(focus)=h(0.5)` です。
